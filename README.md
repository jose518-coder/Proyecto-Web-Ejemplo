# Proyecto-Web-Ejemplo

Cómo clonar y ejecutar el proyecto localmente

Para tener una copia de este proyecto funcionando en tu computadora, sigue estos sencillos pasos:

1. **Clonar el repositorio:**
   Abre tu terminal y ejecuta el siguiente comando:
    git clone https://github.com/jose518-coder/Proyecto-Web-Ejemplo.git

2. **Acceder a la carpeta:** 
    cd Proyecto-Web-Ejemplo

**Ramas**
1. main: Es la rama principal. Contiene el código estable y limpio que se visualiza en producción tras haber resuelto los conflictos de integración.

2. feature-mejora-estilo: Rama de desarrollo utilizada exclusivamente para modernizar la interfaz visual, implementar variables CSS.
